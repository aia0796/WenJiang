## Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository:
    ```bash
    https://github.com/prithwish122/PlayVault.git
    cd my-app
    ```

2. Install the Node.js modules:
    ```bash
    npm install
    ```

### Running Locally

To start the development server and run the project locally:

```bash
npm run dev
```
